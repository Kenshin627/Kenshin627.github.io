еҸ‘иЎЁдәҺ 2022-08-31 еҲҶзұ»дәҺ Computer Graphic пјҢ Math

еӣҫеҪўеӯҰзҡ„ж•°еӯҰеҹәзЎҖпјҲдёүпјүпјҡеҗ‘йҮҸзӮ№з§Ҝ

зӮ№з§Ҝ

зӮ№з§Ҝ($Dot Product$)жҳҜеҗ‘йҮҸд№ҳжі•дёӯжҜ”иҫғз®ҖеҚ•зҡ„дёҖз§Қ(еҸҰеӨ–дёҖз§ҚеҸ«еҒҡеҸүд№ҳ),дҪҶжҳҜзӮ№з§ҜеңЁеӣҫеҪўеӯҰдёӯеә”з”Ёйқһеёёе№ҝжіӣ,дёҺи®ёеӨҡе…¶е®ғиҝҗз®—жңүйҮҚиҰҒзҡ„е…ізі»,дҫӢеҰӮзҹ©йҳөд№ҳжі•гҖҒдҝЎеҸ·еҚ·з§ҜгҖҒз»ҹи®Ўзӣёе…ігҖҒе’ҢеӮ…йҮҢеҸ¶еҸҳжҚўзӯүгҖӮ

е®ҡд№ү

еҗ‘йҮҸзӮ№з§ҜзӯүдәҺдёӨдёӘеҗ‘йҮҸзҡ„й•ҝеәҰзӣёд№ҳеҶҚд№ҳд»ҘдёӨеҗ‘йҮҸеӨ№и§’зҡ„дҪҷејҰ.

$\mathbf{a}\cdot\mathbf{b} = ||\mathbf{a}|| ||\mathbf{b}||\cos{\theta}$

жҺЁеҜј:

$\cos\theta = \dfrac{\mathbf{a}\cdot\mathbf{b}}{||\mathbf{a}|| ||\mathbf{b}||}$

еҜ№дәҺеҚ•дҪҚеҗ‘йҮҸжқҘиҜҙ
$\cos\theta = \mathbf{a}\cdot\mathbf{b}$

е…¬ејҸ

дёӨдёӘеҗ‘йҮҸзҡ„зӮ№з§ҜжҳҜзӣёеә”еҲҶйҮҸзҡ„д№ҳз§Ҝд№Ӣе’ҢпјҢеҫ—еҲ°зҡ„з»“жһңжҳҜдёҖдёӘж ҮйҮҸпјҢеӣ жӯӨд№ҹеҸ«еҒҡж ҮйҮҸд№ҳжі•(Scalar Product)гҖӮ

$\mathbf{a}.\mathbf{b} = \sum\limits_{i=1}^na_ib_i$

дәҢз»ҙе’Ңдёүз»ҙеҗ‘йҮҸзӮ№з§Ҝ

$\mathbf{a}.\mathbf{b} = a_xb_x + a_yb_y$

$\mathbf{a}.\mathbf{b} = a_xb_x + a_yb_y + a_zb_z$

жҖ§иҙЁ

  • дәӨжҚўеҫӢ

    $\mathbf{a}.\mathbf{b} = \mathbf{b}.\mathbf{a}$

  • з»“еҗҲеҫӢ

    $(k\mathbf{a}).\mathbf{b} = \mathbf{a}.(k\mathbf{b}) = k(\mathbf{a}.\mathbf{b})$

  • еҲҶй…ҚеҫӢ

    $\mathbf{a}.(\mathbf{b} + \mathbf{c}) = \mathbf{a}.\mathbf{b} + \mathbf{a}.\mathbf{c}$

еҮ дҪ•ж„Ҹд№ү

еҗ‘йҮҸзӮ№з§Ҝзҡ„еҮ дҪ•ж„Ҹд№үеҜ№дәҺжҲ‘д»¬жқҘиҜҙиҮіе…ійҮҚиҰҒ,еӣ дёәж¶үеҸҠеҲ°еӣҫеҪўеӯҰзҡ„ж–№ж–№йқўйқў,д»ҘдёӢжҲ‘д»¬е°Ҷд»ҺдёӨдёӘж–№йқўиҝӣиЎҢжҺўи®Ё.

жҠ•еҪұ

зӮ№з§Ҝ$\mathbf{a}.\mathbf{b}$зӯүдәҺ$\mathbf{b}$жҠ•еҪұеҲ°е№іиЎҢдәҺ$\mathbf{a}$зҡ„зӣҙзәҝдёҠзҡ„жңүз¬ҰеҸ·еҸ·й•ҝеәҰ,д№ҳд»Ҙ$\mathbf{a}$зҡ„й•ҝеәҰ.
еҰӮдҪ•зҗҶи§ЈиҝҷеҸҘиҜқе‘ў?ж №жҚ®зӮ№з§Ҝзҡ„е®ҡд№үеҸҠзӣҙи§’дёүи§’еҪўдҪҷејҰе®ҡзҗҶжҲ‘д»¬еҫ—зҹҘ

$\mathbf{a}.\mathbf{b} = ||\mathbf{a}|| ||\mathbf{b}||\cos\theta$

->

$||\mathbf{b_вҠҘ}|| = \cos\theta ||\mathbf{b}||$

->

$\mathbf{b_вҠҘ} = ||\mathbf{b_вҠҘ}||\hat{a} = \cos\theta||\mathbf{b}||\hat{a}$

->

$\mathbf{a}\cdot\mathbf{b} = ||\mathbf{a}||||\mathbf{b}|| \cos\theta = ||\mathbf{b_вҠҘ}|| || \mathbf{a}||$

еҲҶи§Јеҗ‘йҮҸ

ж №жҚ®дёҠе°ҸиҠӮеҗ‘йҮҸжҠ•еҪұзҡ„д»Ӣз»ҚжҲ‘д»¬иғҪеӨҹеҫ—еҲ°$\mathbf{b}$еҗ‘йҮҸеңЁ$\mathbf{a}$еҗ‘йҮҸдёҠзҡ„жҠ•еҪұеҗ‘йҮҸ,еӣ жӯӨжҲ‘д»¬еҸҜд»Ҙе°Ҷ$\mathbf{b}$еҗ‘йҮҸиҝӣиЎҢеҲҶи§Ј,е…¶дёӯдёҖжқЎжІҝзқҖ$\mathbf{a}$ж–№еҗ‘еҚі$\mathbf{b_вҠҘ}$,еҸҰеӨ–дёҖжқЎеһӮзӣҙдәҺ$\mathbf{a}$еҗ‘йҮҸ,еҚі$\mathbf{b_вҲҘ}$,еҰӮдёӢеӣҫжүҖзӨә

$\mathbf{b_вҠҘ} = ||\mathbf{b_вҠҘ}|| \hat{a} = \cos\theta ||\mathbf{b}|| \hat{a}$

$\mathbf{b_вҲҘ} = \mathbf{b} - \mathbf{b_вҠҘ}$

еҗ‘йҮҸеӨ§е°ҸдәҺзӮ№з§Ҝзҡ„е…ізі»

$\mathbf{v}.\mathbf{v} = ||\mathbf{v}||^2$

еӨ№и§’

  1. зӮ№з§Ҝзҡ„з¬ҰеҸ·еҸҜд»Ҙз»ҷдәҲжҲ‘д»¬еҜ№дёӨдёӘеҗ‘йҮҸзӣёеҜ№ж–№еҗ‘зҡ„зІ—з•ҘеҲҶзұ»

  1. дёӨдёӘеҚ•дҪҚеҗ‘йҮҸзҡ„зӮ№з§ҜзӯүдәҺеӨ№и§’зҡ„дҪҷејҰ.

    $\hat{a}\cdot\hat{b} = \cos\theta$

  2. дҪҝз”ЁзӮ№з§Ҝи®Ўз®—дёӨдёӘеҗ‘йҮҸд№Ӣй—ҙзҡ„и§’еәҰ

    $\theta = \arccos(\dfrac{\mathbf{a}\cdot\mathbf{b}}{||\mathbf{a}||||\mathbf{b}||})$

гҖҠ3Dж•°еӯҰеҹәзЎҖгҖӢеӣҫеҪўе’ҢжёёжҲҸејҖеҸ‘(з¬¬дәҢзүҲ)
